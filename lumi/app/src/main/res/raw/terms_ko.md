# 루미 이용약관

개정일: 2026년 5월 18일
시행일: 2026년 5월 18일

본 약관은 루미(이하 "서비스")의 이용 조건과 사용자 및 운영자의 권리·의무를 정합니다. 사용자는 본 약관, 개인정보 처리방침, 법적 고지에 동의한 뒤 서비스를 이용할 수 있습니다.

## 1. 서비스의 내용

1.1 서비스는 감정 기반 AI 캐릭터와의 대화, 대화 기록 저장, 루미의 상태·기억 표시, 능동 대화 알림, 음성 인식 입력, 이미지 첨부, 이미지 생성, 대화 내보내기 기능을 제공합니다.

1.2 서비스는 사용자의 단말에 저장된 데이터와 사용자가 입력한 설정을 기반으로 동작합니다. 원격 LLM 또는 이미지 API를 켠 경우 사용자가 선택한 외부 API 제공자의 서비스를 호출할 수 있습니다.

1.3 서비스는 의료, 법률, 금융, 투자, 심리 상담, 긴급 구조, 안전 보장 서비스를 제공하지 않습니다. 중요한 판단은 반드시 전문가 또는 공식 기관의 정보를 확인해야 합니다.

## 2. 이용 자격 및 동의

2.1 서비스는 만 14세 이상 사용자를 대상으로 합니다.

2.2 미성년자는 법정대리인의 동의를 받은 경우에만 서비스를 이용할 수 있습니다.

2.3 약관 또는 개인정보 처리방침이 중요한 범위에서 변경되는 경우, 서비스는 약관 버전을 올려 사용자의 재동의를 요구할 수 있습니다.

2.4 사용자는 설정 또는 법적 고지 화면에서 동의를 철회할 수 있습니다. 필수 동의를 철회하면 서비스를 계속 이용하려면 다시 동의해야 합니다.

## 3. 사용자 책임

3.1 사용자는 불법, 차별, 명예훼손, 괴롭힘, 성착취, 아동·청소년 유해, 폭력 조장, 타인의 개인정보 침해, 저작권 침해 목적의 내용을 입력하거나 생성 요청해서는 안 됩니다.

3.2 사용자가 입력한 대화, 이미지 프롬프트, 첨부 이미지, API 키, 모델명, 엔드포인트 URL, 내보낸 파일의 관리 책임은 사용자에게 있습니다.

3.3 사용자가 외부 API 키를 직접 입력해 사용하는 경우, 해당 API 제공자의 약관, 요금, 사용량 제한, 콘텐츠 정책, 데이터 처리 정책이 함께 적용됩니다.

3.4 사용자는 자신 또는 타인의 주민등록번호, 계좌번호, 인증번호, 비밀번호, 민감한 건강정보 등 꼭 필요하지 않은 민감정보를 입력하지 않아야 합니다.

## 4. 외부 AI/API 및 웹 검색

4.1 원격 LLM 기능을 켠 경우 서비스는 대화 응답 생성을 위해 시스템 프롬프트, 최근 대화 일부, 루미의 상태·기억 요약, 사용자 입력을 사용자가 선택한 API 제공자에게 전송할 수 있습니다.

4.2 이미지 생성 기능을 켠 경우 서비스는 이미지 프롬프트와 설정값을 이미지 API 제공자에게 전송하고, 생성된 이미지를 앱 캐시 또는 사용자가 선택한 저장 위치에 저장할 수 있습니다.

4.3 최신 정보, 검색, 뉴스, 공식 정보 등을 묻는 입력이 있는 경우 서비스는 검색 결과를 보강하기 위해 웹 검색 요청 및 검색 결과 페이지 요약 요청을 수행할 수 있습니다.

4.4 외부 API 제공자에는 OpenAI, Anthropic, Google Gemini, OpenAI 호환 API 제공자, 이미지 API 제공자, 웹 검색 또는 검색 결과 페이지 제공자가 포함될 수 있습니다.

4.5 외부 API 또는 웹 검색 사용으로 발생하는 요금, 장애, 지연, 정책 변경, 계정 제한, 결과 품질 문제는 각 제공자의 정책과 사용자 설정에 따릅니다.

## 5. 결제 및 비용

5.1 현재 서비스는 앱 자체 결제 기능을 포함하지 않습니다.

5.2 사용자가 직접 입력한 외부 LLM·이미지 API 키로 호출하는 비용은 해당 API 계정 소유자에게 청구될 수 있습니다.

5.3 향후 구독, 인앱결제, 유료 기능이 추가되는 경우 별도의 결제 조건과 환불 기준을 고지하고 필요한 동의를 받습니다.

## 6. 콘텐츠와 지식재산권

6.1 사용자가 입력한 텍스트, 첨부 이미지, 내보낸 대화 파일에 대한 권리는 원칙적으로 사용자에게 있습니다.

6.2 루미의 앱 이름, 캐릭터, 디자인, 코드, 화면 구성, 기본 프롬프트, 로컬 응답 규칙 등 서비스 구성 요소의 권리는 운영자 또는 정당한 권리자에게 있습니다.

6.3 AI가 생성한 텍스트·이미지의 권리와 이용 가능 범위는 관련 법령 및 외부 API 제공자의 약관에 따라 달라질 수 있습니다. 상업적 이용 전 사용자는 권리 관계를 직접 확인해야 합니다.

## 7. 서비스 변경 및 중단

7.1 서비스는 기능 개선, 보안, 법령 준수, 외부 API 정책 변경, 운영상 필요에 따라 기능을 변경하거나 일부 제공을 중단할 수 있습니다.

7.2 단말 상태, 네트워크 상태, 외부 API 장애, OS 정책, 권한 설정에 따라 일부 기능이 정상적으로 작동하지 않을 수 있습니다.

## 8. 책임의 한계

8.1 서비스는 AI 기반 보조 도구이며 응답의 정확성, 완전성, 최신성, 특정 목적 적합성을 보장하지 않습니다.

8.2 사용자가 AI 응답, 알림, 이미지 생성 결과, 검색 결과를 신뢰하여 한 행위나 손해에 대해 운영자는 고의 또는 중대한 과실이 없는 한 책임을 지지 않습니다.

8.3 외부 API 제공자, 검색 결과 페이지, OS 음성 인식 서비스, 갤러리·공유 앱 등 제3자 서비스에서 발생한 문제는 각 제공자의 정책에 따릅니다.

## 9. 약관 변경

9.1 중요한 약관 변경이 있는 경우 앱 내 고지 또는 재동의 화면으로 안내합니다.

9.2 사용자가 변경 약관에 동의하지 않으면 서비스를 계속 이용할 수 없으며, 설정에서 데이터를 삭제하거나 앱을 삭제할 수 있습니다.

## 10. 준거법 및 분쟁 해결

10.1 본 약관은 대한민국 법령에 따라 해석됩니다.

10.2 분쟁이 발생하는 경우 대한민국 민사소송법 등 관련 법령에 따른 관할 법원에서 해결합니다.

## 11. 문의

서비스 관련 문의는 앱 스토어 등록 페이지 또는 운영자가 별도로 고지한 고객지원 채널을 통해 접수합니다.
