# 루미 개인정보 처리방침

개정일: 2026년 5월 18일
시행일: 2026년 5월 18일

루미(이하 "서비스")는 사용자의 대화와 설정을 단말 중심으로 처리합니다. 본 방침은 서비스가 어떤 정보를 어떤 목적으로 처리하는지 설명합니다.

## 1. 처리하는 정보

서비스는 다음 정보를 처리할 수 있습니다.

1.1 사용자가 직접 입력한 정보
- 대화 내용, 이미지 생성 요청 문장, 첨부 이미지 URI 또는 경로, 음성 인식 결과 텍스트
- 사용자 이름 또는 호칭
- LLM 제공자, 모델명, 기본 URL, OpenAI 호환 엔드포인트
- LLM API 키, 이미지 API 키, Google Search API 키 및 검색 엔진 ID를 포함한 연결 설정
- 이미지 생성 활성화 여부, NSFW 허용 여부, 능동 대화 알림 설정

1.2 서비스 이용 과정에서 생성되는 정보
- 루미의 성격·기분·친밀도·성장 단계
- 루미가 추출한 기억 키워드와 대화 요약성 데이터
- 대화 시각, 일일 대화 횟수, 능동 대화 발송 횟수
- 생성 이미지 파일, 이미지 캐시, 사용자가 저장·공유한 이미지의 URI
- 대화 내보내기 파일 내용

1.3 권한 기반 정보
- 마이크 권한: 음성 인식 입력을 위해 사용합니다. 앱은 원본 음성 파일을 별도로 저장하지 않고 인식된 텍스트를 대화로 저장합니다.
- 알림 권한: 루미의 능동 대화 알림을 표시하기 위해 사용합니다.
- 인터넷 권한: 원격 LLM, 이미지 API, 웹 검색, 검색 결과 페이지 확인에 사용합니다.

## 2. 이용 목적

서비스는 정보를 다음 목적으로 이용합니다.

- AI 대화 및 로컬 응답 생성
- 원격 LLM 응답 생성
- 이미지 생성, 이미지 캐시 저장, 갤러리 저장 및 공유
- 사용자의 대화 맥락 유지, 루미의 기억·성격·친밀도 갱신
- 음성 입력을 텍스트로 변환해 대화에 반영
- 최신 정보 요청 시 웹 검색 및 검색 결과 요약
- 대화 내역 내보내기
- 서비스 안정성, 권한 안내, 설정 저장

## 3. 보관 위치와 기간

3.1 대화, 기억, 루미 상태, 생성된 일부 데이터는 사용자의 단말 내 앱 데이터베이스(Room)에 저장됩니다.

3.2 API 키는 Android Keystore 기반 EncryptedSharedPreferences에 암호화 저장하는 것을 우선합니다. 단말 보안 저장소 오류 등으로 암호화 저장소 초기화에 실패하는 경우 앱 전용 저장소로 폴백할 수 있습니다.

3.3 생성 이미지는 앱 캐시에 저장되며 오래된 캐시는 앱이 가능한 범위에서 정리합니다. 사용자가 갤러리에 저장하거나 다른 앱으로 공유한 이미지는 해당 저장 위치 또는 공유 앱의 정책을 따릅니다.

3.4 사용자가 설정에서 대화, 기억, 전체 데이터를 초기화하면 해당 범위의 앱 내부 데이터가 삭제됩니다.

3.5 앱을 삭제하면 Android OS 정책에 따라 앱 내부 저장소의 데이터가 삭제됩니다. 사용자가 별도로 내보낸 파일, 갤러리에 저장한 이미지, 다른 앱에 공유한 자료는 앱 삭제로 삭제되지 않을 수 있습니다.

## 4. 외부 전송 및 국외 이전

서비스는 자체 운영 서버로 사용자의 데이터를 자동 업로드하지 않습니다. 다만 사용자가 원격 기능을 켜거나 관련 기능을 실행하면 다음 정보가 외부 제공자에게 전송될 수 있습니다.

4.1 원격 LLM
- 전송 항목: 시스템 프롬프트, 최근 대화 일부, 사용자 입력, 루미의 상태·기억 요약, 모델 설정
- 제공자 예시: OpenAI, Anthropic, Google Gemini, OpenAI 호환 API 제공자
- 이전 국가: 제공자 정책에 따르며 미국 등 국외가 포함될 수 있습니다.
- 보관 기간: 각 제공자의 약관 및 개인정보 정책에 따릅니다.

4.2 이미지 생성 API
- 전송 항목: 이미지 프롬프트, 모델명, 이미지 API 설정
- 제공자 예시: OpenAI Images, OpenAI 호환 이미지 API 제공자
- 보관 기간 및 처리 기준: 각 제공자 정책에 따릅니다.

4.3 웹 검색 및 검색 결과 페이지
- 전송 항목: 사용자가 최신 정보·검색·공식 정보 등을 요청한 문장 또는 검색어
- 제공자 예시: DuckDuckGo HTML 검색, 검색 결과로 연결되는 웹사이트
- 목적: 검색 결과 제목·요약·페이지 설명 확인

4.4 음성 인식
- 서비스는 Android SpeechRecognizer를 사용합니다. 단말 설정과 OS 음성 인식 제공자에 따라 음성 데이터가 기기 내에서 처리되거나 Google 등 음성 인식 제공자의 서버로 전송될 수 있습니다.
- 앱은 원본 음성 파일을 저장하지 않으며, 인식 결과 텍스트를 대화 기록으로 저장합니다.

4.5 외부 API 키
- API 키는 요청 인증을 위해 해당 API 제공자에게 전송됩니다.
- 사용자는 설정에서 API 키를 삭제하거나 원격 기능을 끌 수 있습니다.

## 5. 제3자 제공 및 처리 위탁

서비스는 운영자 자체 서버를 통한 제3자 제공을 하지 않습니다. 외부 API 또는 OS 음성 인식 기능은 사용자의 설정과 동작에 따라 사용자가 선택하거나 단말이 제공하는 제3자 서비스로 연결되는 것입니다. 이 경우 해당 제3자의 약관과 개인정보 정책이 적용됩니다.

## 6. 안전성 확보 조치

- 앱 내부 데이터는 Android 앱 샌드박스 영역에 저장합니다.
- API 키는 가능한 경우 Android Keystore 기반 암호화 저장소에 보관합니다.
- Android 백업 및 기기 간 전송에서 대화 DB, 설정, API 키, 음성 관련 파일을 제외하도록 설정합니다.
- 외부 통신은 HTTPS 기반 API를 사용합니다.
- 이미지 응답 크기에 제한을 두고 캐시를 주기적으로 정리합니다.

## 7. 사용자 권리

사용자는 다음 권리를 행사할 수 있습니다.

- 열람: 앱 내 대화, 상태, 기억 화면에서 확인
- 정정: 대화 재입력, 설정 변경, 사용자 이름 변경
- 삭제: 설정의 대화 초기화, 성장 데이터 초기화, 전체 초기화 또는 앱 삭제
- 동의 철회: 법적 고지 및 정책 화면의 동의 철회 기능 사용
- 이동: 대화 내보내기 기능으로 텍스트 파일 저장

## 8. 만 14세 미만 아동

서비스는 만 14세 이상을 대상으로 합니다. 만 14세 미만 사용자의 정보가 처리된 사실을 인지한 경우 운영자는 가능한 범위에서 해당 정보 삭제 또는 이용 제한 조치를 취합니다.

## 9. 개인정보 보호 문의

개인정보 관련 문의는 앱 스토어 등록 페이지 또는 운영자가 별도로 고지한 고객지원 채널을 통해 접수합니다.

## 10. 변경 고지

본 방침이 중요한 범위에서 변경되는 경우 앱 내 고지 또는 재동의 화면을 통해 안내합니다.
