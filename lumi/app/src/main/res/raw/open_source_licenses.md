# 오픈소스 라이선스

루미는 다음 오픈소스 소프트웨어를 사용합니다. 각 라이브러리의 전체 라이선스 전문은 해당 프로젝트의 공식 저장소, Maven 배포 메타데이터 또는 Android Open Source Project 고지에서 확인할 수 있습니다.

## Android Platform APIs

앱은 Android SDK가 제공하는 SpeechRecognizer, Notification, FileProvider, SharedPreferences, Room 기반 저장소 등 Android 플랫폼 API를 사용합니다. Android 플랫폼 구성 요소의 라이선스와 고지는 Android Open Source Project 및 Android SDK 배포 고지를 따릅니다.

## 고지

외부 LLM/API 제공자(OpenAI, Anthropic, Google Gemini, OpenAI 호환 API 제공자 등)는 오픈소스 라이브러리가 아니며, 각 제공자의 별도 약관과 정책이 적용됩니다.
