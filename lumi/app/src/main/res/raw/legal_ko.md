# 법적 고지

개정일: 2026년 5월 18일
시행일: 2026년 5월 18일

## 1. AI 서비스 고지

루미는 생성형 AI 및 로컬 규칙 기반 응답을 함께 사용하는 보조 애플리케이션입니다. AI 응답은 부정확하거나 오래되었거나 사용자의 의도와 다를 수 있습니다. 중요한 결정에는 공식 자료와 전문가 판단을 함께 확인해야 합니다.

## 2. 외부 서비스 고지

사용자가 원격 LLM, 이미지 생성, 웹 검색, 음성 인식 기능을 사용하는 경우 OpenAI, Anthropic, Google, OpenAI 호환 API 제공자, DuckDuckGo, 검색 결과 웹사이트, Android 음성 인식 제공자 등 외부 서비스가 관여할 수 있습니다.

외부 서비스의 장애, 지연, 요금, 정책 변경, 계정 제한, 콘텐츠 필터링, 데이터 보관 기준은 각 제공자의 약관과 정책에 따릅니다.

## 3. 비용 고지

루미 앱 자체에는 현재 인앱결제 기능이 없습니다. 다만 사용자가 외부 API 키를 입력해 원격 LLM 또는 이미지 생성 API를 호출하면 해당 API 제공자가 사용량에 따라 비용을 청구할 수 있습니다.

## 4. 데이터 처리 고지

루미는 자체 운영 서버로 데이터를 자동 업로드하지 않습니다. 기본 데이터는 단말 내부에 저장됩니다. 그러나 원격 기능을 켠 경우 대화, 프롬프트, 검색어, 모델 설정 등이 외부 제공자에게 전송될 수 있습니다.

## 5. 긴급 상황 고지

루미는 응급 구조, 자해 예방 상담, 의료·심리 위기 대응 서비스를 제공하지 않습니다. 긴급하거나 생명·안전과 관련된 상황에서는 즉시 119, 112, 지역 응급기관, 의료기관, 보호자 등 실제 도움을 받을 수 있는 경로를 이용해야 합니다.

## 6. 오픈소스 고지

루미는 AndroidX, Material Components, Room, AndroidX Security 등 오픈소스 소프트웨어를 사용합니다. 각 라이선스 조건은 오픈소스 라이선스 화면에서 확인할 수 있습니다.

## 7. 운영자 정보

문의와 권리 행사 요청은 앱 스토어 등록 페이지 또는 운영자가 별도로 고지한 고객지원 채널을 통해 접수합니다.
