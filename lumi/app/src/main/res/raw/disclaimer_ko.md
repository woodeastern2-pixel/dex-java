# 책임 한계 고지

개정일: 2026년 5월 18일
시행일: 2026년 5월 18일

## 1. AI 응답의 한계

루미의 응답은 생성형 AI 또는 로컬 규칙 기반 로직으로 만들어질 수 있습니다. 응답은 사실과 다를 수 있고, 최신 정보가 아닐 수 있으며, 특정 목적에 적합하지 않을 수 있습니다.

## 2. 전문 자문 아님

루미는 의료, 법률, 금융, 투자, 세무, 심리 상담, 긴급 구조, 안전 판단을 대체하지 않습니다. 사용자는 중요한 결정을 내리기 전에 전문가, 공식 기관, 원자료를 확인해야 합니다.

## 3. 감정·관계형 대화 고지

루미는 사용자의 감정을 배려하는 캐릭터형 앱이지만 실제 인간, 의료인, 상담사, 법률 전문가가 아닙니다. 지속적인 우울, 자해 충동, 폭력 위험, 긴급한 안전 문제가 있는 경우 즉시 119, 112, 의료기관, 보호자, 지역 상담기관 등 실제 도움을 받을 수 있는 경로를 이용해야 합니다.

## 4. 외부 서비스 책임

원격 LLM, 이미지 생성 API, 웹 검색, OS 음성 인식, 갤러리 저장, 공유 앱 등 외부 서비스에서 발생하는 장애, 비용, 정책 변경, 콘텐츠 제한, 데이터 처리 문제는 각 제공자의 약관과 정책에 따릅니다.

## 5. 비용과 API 키

사용자가 직접 입력한 API 키로 발생한 요금, 사용량 초과, 계정 정지, 키 유출, 모델 선택 오류는 사용자의 API 계정 관리 범위에 속합니다. 루미는 API 제공자의 청구 또는 환불을 대신 처리하지 않습니다.

## 6. 생성 콘텐츠 책임

사용자가 입력한 프롬프트와 생성·저장·공유한 텍스트 또는 이미지의 이용 책임은 사용자에게 있습니다. 타인의 권리 침해, 명예훼손, 불법 콘텐츠 생성 및 배포는 금지됩니다.

## 7. 손해 배상 제한

관련 법령상 허용되는 범위에서, 운영자는 고의 또는 중대한 과실이 없는 한 서비스 이용 또는 이용 불가로 발생한 간접손해, 특별손해, 결과손해, 데이터 손실, 영업손실, 외부 API 비용에 대해 책임을 지지 않습니다.
